# Git コマンド

```
// Gitの準備
Git init
```

```
// ファイルを追加
git add index.html
```

```
// ファイルの記録(メッセージはわかりやすく書く！！)
git commit -m"メッセージ"
```

```
// リモートリポジトリ登録
git remote add リモートリポジトリ名 URL
```

```
// リモートリポジトリにファイルをアップロード
git push origin master
```

```
// リポジトリのファイルをダウンロード
git pull origin master
```

```
// 変更したファイル名を確認
git status
```

```
// 変更した内容を確認
git diff
```

```
// gitのログを見る
git log
// gitの変更内容を見る-->Qキーで終了
git log -p

```

# 共有の仕組み

- Git ではリモートまたはリモートリポジトリで共有ファイルの置き場を使う

# Git の大事なこと

- 自分が行った変更を把握してその変更の中で相手に共有すべき部分を選択できるようにすること
