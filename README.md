# llm_learning_journey

技術理解を深めるための学習リポジトリです（Web・インフラ・データベースなど、言語を問わない基礎概念の学習記録を含みます）。あわせて、LLM・生成AI関連のポートフォリオ作品も含みます。

## ポートフォリオ

- **[TechDoc QA Bot](./03_portfolio/techdoc-qa-bot/)** — 技術ドキュメント(PDF)に自然言語で質問できるRAG構成のQAツール(Python / LangChain / ChromaDB / Claude API)

## リポジトリ構成

```text
01_basics/       Git・コマンドライン・Python・SQLなど、言語・分野を問わない基礎学習
                 (concepts/ は技術理解を深めるための学習ノート置き場)
02_speciality/   Docker・FastAPI・LangChainなどの専門技術の学習
03_portfolio/    ポートフォリオ作品
```
